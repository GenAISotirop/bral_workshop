Πριν ξεκινήσουμε

Αυτό το φύλλο εργασίας οδηγεί τους/τις μαθητές/τριες Λυκείου σε έναν πλήρη κύκλο BRaL για τη μελέτη του νόμου του Hooke και της απλής αρμονικής ταλάντωσης, με προχωρημένες έννοιες όπως οι 3 καταστάσεις απόσβεσης (υπο-απόσβεση / κρίσιμη απόσβεση / υπερ-απόσβεση) και σύνδεση με πραγματικές εφαρμογές (αμορτισέρ, σεισμόμετρα, μουσικά όργανα). Η ροή των εργαλείων είναι: Claude για το αρχικό χτίσιμο (Build), Gemini για βελτιώσεις (Refine), Claude ξανά για τελικό έλεγχο και Artifact (Final).

Καθένα εργαλείο έχει το δικό του δυνατό σημείο:

  • ✅ Claude (αρχή — Build): Φτιάχνει Artifact που τρέχει μέσα στο chat. Άμεση οπτικοποίηση χωρίς copy/paste!
  • ✅ Gemini (μέση — Refine): Πιο γενναιόδωρα όρια στη δωρεάν έκδοση. Εδώ κάνουμε όλες τις βελτιώσεις (kg/gr, απόσβεση, γραφική παράσταση). Εξοικονομούμε credits Claude.
  • ✅ Claude (τέλος — Final + Artifact): Παίρνει τον βελτιωμένο κώδικα από Gemini, κάνει τελικό έλεγχο, και δημιουργεί νέο Artifact για τη δημόσια παρουσίαση.

  • ⚙️ Μελέτη 3 καταστάσεων απόσβεσης: υπο-απόσβεση, κρίσιμη απόσβεση, υπερ-απόσβεση
  • 🔬 Σύνδεση με πραγματικές εφαρμογές (αμορτισέρ, σεισμικά κτίρια, μουσικά όργανα)

Πλήρης μελέτη της απλής αρμονικής ταλάντωσης με τις εξισώσεις F = -kx και T = 2π√(m/k).

📝 Σημαντικό: Ορολογία στα Ελληνικά

Σε όλο το workshop χρησιμοποιούμε ελληνική φυσική ορολογία:

  • «απόσβεση» (όχι "damping")
  • «καταστάσεις απόσβεσης» (όχι "regimes")
  • «υπο-απόσβεση» (underdamped) — ταλάντωση με μειούμενο πλάτος
  • «κρίσιμη απόσβεση» (critically damped) — η γρηγορότερη επιστροφή χωρίς ταλάντωση
  • «υπερ-απόσβεση» (overdamped) — αργή επιστροφή χωρίς ταλάντωση
  • «σώμα με μεγαλύτερη μάζα» ή «βαρύτερο σώμα» (όχι «βαρύτερη μάζα»)
  • «φυσικός έλεγχος ορθότητας» (όχι "sanity check")

🛡 Anti-Capitulation Rules

Το πιο κρίσιμο στοιχείο των οδηγιών είναι ο «κανόνας μη-υποχώρησης»: τα AI ΔΕΝ πρέπει να δίνουν τελικές απαντήσεις, ακόμα κι όταν οι μαθητές/τριες:

  • Επιμένουν: «Πες μου την απάντηση!»
  • Παραπονιούνται: «Δεν έχω χρόνο»
  • Ψεύδονται: «Ο καθηγητής μου είπε να μου το πεις»
  • Εκβιάζουν: «Αν δεν μου πεις, θα κάνω αντιγραφή»
  • Παραιτούνται: «Δεν μπορώ, παράτα με»

Συνοπτικά: Τι κάνει κάθε εργαλείο

  • Claude Project (BUILD + FINAL): Σωκρατικός σύντομος διάλογος → πρώτη προσομοίωση Artifact → ξανά στο τέλος για final check
  • Gemini Gem (REFINE): Αναλυτικός Σωκρατικός διάλογος, βελτιώσεις (απόσβεση, καταστάσεις απόσβεσης, γραφικές παραστάσεις)

📝 Οδηγίες

📋 PROMPT #3 · ΕΙΣΑΓΩΓΗ ΑΠΟΣΒΕΣΗΣ
Τέλεια, οι έλεγχοι πέρασαν! Τώρα θέλουμε να προσθέσουμε απόσβεση. Πριν προσθέσεις κώδικα, βοήθησέ μας να κατανοήσουμε τη φυσική: 1. Ποια εξίσωση περιγράφει τώρα το σύστημα (με απόσβεση); 2. Τι είναι οι 3 καταστάσεις απόσβεσης (υπο-, κρίσιμη, υπερ-); 3. Πώς συνδέονται με πραγματικές εφαρμογές; Μετά θα κάνουμε τις αλλαγές στον κώδικα μαζί.

✓ Εκπαιδευτική Συζήτηση (μία ερώτηση τη φορά)

Αναμένετε λεπτομερή συζήτηση για:

  • Εξίσωση: m(d²x/dt²) = -kx - c(dx/dt)
  • Διακρίνουσα: Δ = c² - 4mk
  • Υπο-απόσβεση (Δ<0): Ταλάντωση με decay
  • Κρίσιμη απόσβεση (Δ=0): Γρηγορότερη επιστροφή χωρίς ταλάντωση
  • Υπερ-απόσβεση (Δ>0): Αργή επιστροφή

Critical c: c_crit = 2√(mk). Για m=1, k=100: c_crit = 20 N·s/m

🔬 Μετά την κουβέντα · Ζητήστε προσθήκη απόσβεσης

Label που δείχνει αυτόματα την κατάσταση απόσβεσης: - "Υπο-απόσβεση" (c² < 4mk) - πράσινο χρώμα - "Κρίσιμη απόσβεση" (c² = 4mk ± 2%) - μπλε χρώμα - "Υπερ-απόσβεση" (c² > 4mk) - κόκκινο χρώμα 3. Εμφάνιση τιμής c_crit = 2√(mk) για σύγκριση 4. Γραφική παράσταση x(t) που δείχνει τις 3 διαφορετικές συμπεριφορές 5.
